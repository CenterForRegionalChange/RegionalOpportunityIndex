About the Data

A full description of our data sources and methodology for calculating the ROI and its various components is available here.

A full version of the data mapped on this site, including the original indicators and calculated indices, is available for download in excel format here.

While we have selected what we believe is the best available data sources for our particular indicators, each dataset has certain limitations that are important to keep in mind when using them to analyze conditions in any location. Before using the data, please read the section of the document that describes data limitations and how we have handled them in our index and indicator construction. If you discover any inaccuracies, please inform us by emailing swatterson@ucdavis.edu.
