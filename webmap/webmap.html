Use this tool to compare across indicators and census tracts. To use this feature, all three maps must contain census tract data. Clicking on additional tracts will add their indicator data to each of the three tables below. Clicking the "Clear" button, or changing the display layer in any of the three maps, will clear the results and allow you to start over. Clicking the "CSV" button will allow these results to be saved in a tabular format, with the three indicators as columns and the tracts as rows. These on-screen table results are formatted as Tract: Value.
